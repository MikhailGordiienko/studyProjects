Task 2.2-1

Вывести название, жанр и цену тех книг, количество которых больше 8,
в отсортированном по убыванию цены виде.

My code

SELECT title, name_genre, price
FROM
    book INNER JOIN genre
    ON book.genre_id = genre.genre_id
WHERE amount > 8
ORDER BY price DESC;
