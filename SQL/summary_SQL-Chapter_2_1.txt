Task 2.1-5

Создать таблицу author следующей структуры:

Поле	     Тип, описание
author_id	 INT PRIMARY KEY AUTO_INCREMENT
name_author	 VARCHAR(50)

My code

CREATE TABLE author (
    author_id	INT PRIMARY KEY AUTO_INCREMENT,
    name_author	VARCHAR(50)
);
